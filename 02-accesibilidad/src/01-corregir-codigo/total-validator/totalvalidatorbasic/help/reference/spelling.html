Spell check problems

These problems are due to spelling mistakes or language problems
CodeSummaryDetails
E031Spelling mistakesWords that are not found in the dictionary are highlighted followed by a list of suggested replacements in brackets. To correct this warning replace the word with one of the suggestions or an alternative word. If you believe the word to be correct then with Total Validator Pro you can add the word to a user-defined dictionary by clicking on it as described in the help.
E032Document language doesn't match the spell check languageIf the document language isn't the same as the language set for spell checking then the document will not be spell checked. So you may have spelling errors on this page. If this wasn't intentional either change the language used for spell checking or correct the language of the document.
